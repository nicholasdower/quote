# quote

## Install

```shell
brew install nicholasdower/tap/quote
```

## Uninstall

```shell
brew uninstall quote
```

## Help

```
usage:
    quote
    quote <quote>
    quote <start-quote> <end-quote>

Description

    Quote lines, optionally using the specified quote character(s).

Options

    -h, --help       Print help.
    -v, --version    Print version.
```
