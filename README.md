# quote

```
usage: quote ([-q <quote>] | [-s <start>] [-e <end>]) [file ...]

Quote lines

Options

    -q, --quote      The quote character.
    -s, --start      The staring quote character.
    -e, --end        The ending quote character.
    -h, --help       Print help.
    -v, --version    Print version.
```
